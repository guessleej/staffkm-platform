staffKM Embeddable Widget

一行 <script> 把 chat 嵌任何網頁。下方右下角應有 「有問題嗎？」 浮動按鈕。

📋 嵌入 snippet

<script src="/widget.js"
        data-app-id="YOUR_APP_UUID"
        data-position="bottom-right"
        data-color="#4f46e5"
        data-label="有問題嗎？"
        defer></script>

必填只有 data-app-id；application 必須 is_public = true。

🛠 互動測試

或在 console 跑：staffKM.open() / staffKM.close()

📝 屬性

屬性說明預設
data-app-id必填，your public app UUID—
data-hoststaffKM domainscript src origin
data-positionbottom-right / bottom-leftbottom-right
data-colorlauncher 底色（任意 CSS color）#4f46e5
data-labellauncher 文字有問題嗎？
