# 對話與知識庫

## 對話技巧

- **直接問**：「請假怎麼申請」比「請問貴公司關於休假相關規定的申請流程」更好
- **追問**：可以接著問「需要主管簽嗎」「最多可以請幾天」，staffKM 會記得脈絡
- **點 citation**：每段回答下方會列出引用文件，點進去看原文

## 切換 application

不同 application 接不同 KB / model：

- 全公司 SOP → 「公司知識」application
- 採購流程 → 「採購助理」application
- 部門專屬 → 部門自己的 application（PM 建立）

對話頁右上有切換器。

## ⌘K 快速跳轉

任何頁面按 `⌘K`（Mac）/ `Ctrl K`（Windows）：
- 輸入 `app` 跳到應用列表
- 輸入 `用量` 跳到 token dashboard（admin only）
- ↑↓ 選擇 + Enter 執行

## 我的對話會被誰看到？

- **預設**：只有你自己看得到
- **app 公開分享**：admin 可以為某個 application 產生 share link 給外部（不需要帳號）
- **workspace 內**：admin 可看 audit log（公司管理需求）
