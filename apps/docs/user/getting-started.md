# 5 分鐘上手

## 1. 登入
打開 `https://staffkm.<你的網域>`，用公司 SSO 或 admin 給的帳號登入。

## 2. 選 workspace
右上角下拉選擇你被分配到的 workspace。若只有一個會自動選。

## 3. 開始對話
左側 nav 點「對話」→ 直接打字。staffKM 會找最近相關的內部文件回答。

> **進階**：按 `⌘K` 開命令面板，輸入 `app` 切到應用列表，輸入 `usage` 看用量。

## 4. 看 citation
回答下方會列出引用文件。點進去看原文。

## 5. 建立你自己的 application（PM / Editor 角色）

「應用」→ 新增 → 簡易問答 / 工作流：
- **簡易問答**：選一個 LLM model + 一個 KB，搞定
- **工作流**：拖拉節點（LLM / 知識庫 / 條件 / HTTP 等）

設定完按右上「發布」即可被同 workspace 的同事使用。
