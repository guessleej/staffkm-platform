# v5.11 線 → v5.12.0 consolidation

> v5.11.0–v5.11.22（23 個 tag）的整合里程碑。無破壞性 schema 變更；皆為加法層 + 地端
> 模型治理 + 測試硬化。本檔給升級者快查「做了什麼、要設什麼、注意什麼」。

## 四大主題

### 1. GraphRAG（RFC-014）— 加法層，MVP → 進階
- **MVP（v5.11.0）**：ingest 時用地端 LLM 抽實體 + 字串匹配建 grounded mention；query 時純向量
  比對 `kb_entities`（不呼叫 LLM）→ RRF 第三路融進 hybrid。圖只做「檢索/路由」，答案永遠出自
  原始段落（數字不失真）。混合語料分流（`looks_like_form`）：表單/收據跳過、散文跑圖。
- **召回調校（v5.11.4）**：根因是 `ivfflat.probes=1`（非融合權重）→ 設 `IVFFLAT_PROBES=10`；
  加 graph-only cosine 門檻。A/B：recall@5 base 0.40 → graph 0.88、0 退步。
- **Phase 1（v5.11.16）**：持久化實體關係 `kb_relations` + `kb_relation_mentions`（co-occurrence grounded）。
- **Phase 2（v5.11.17）**：多跳召回（沿關係擴 1 跳）。**預設 `GRAPH_QUERY_HOPS=0`** —— 小語料增益=0
  （top-5 錨定已涵蓋、文件內共現鄰居共用段落），evidence-gated，語料變大再開。
- **Phase 3（v5.11.18）**：community detection（連通分量分群 + 每社群地端 LLM 摘要）→ `kb_communities`；
  `GET /knowledge/bases/{kb}/graph/communities`。

### 2. 地端模型治理（不受雲端內容審查 / 不寫死）
- Ollama provider 指 `host.docker.internal:11434`；`/admin/models` 模型清單**動態同步 `/api/tags`**（不寫死）。
- 系統預設模型（`system_settings.default.*`）**接到 runtime**：`default.llm`（聊天）/`default.vision`（OCR）/
  `default.rerank`（重排）選了即時生效（CLAUDE.md §11）。`default.stt/tts`=planned（無 pipeline，UI 標示）。
- Big5/CP950 編碼偵測（台灣公文/舊 Excel 不再亂碼）。

### 3. embedding 熱換（v5.11.19）
- 雙設定：`default.embedding`(desired) vs `embedding.active`(已套用)。runtime 一律讀 active（query↔索引一致）。
- 換模型 = 全庫重嵌：`POST /knowledge/embedding/reindex {model_name}` → 維度不符時遷移兩個共用 vector 欄
  (`paragraph_embeddings`+`kb_entities`) → 重嵌 → 重建 ivfflat → 寫 `embedding.active`。`/admin/models`
  有「套用（全庫重新索引）」按鈕 + 進度。**重嵌期間搜尋退化、請勿上傳新文件**。

### 4. 測試/CI 硬化（v5.11.5–15）
- E2E（ingest→檢索）、SSE 換行根因修 + 多層守衛、RAG retrieval 回歸 gate、Playwright 對話渲染。
- landmine 守衛：asyncpg cast / array、advisory 未接通、Decimal+float、ollama `/v1` 契約、幽靈模型、
  `async with get_session()`、**新 Celery task 必有 queue 路由**。
- 3 個重測 workflow 加 nightly cron（gated）。

## 升級者要設 / 注意
| 項目 | 動作 |
|---|---|
| 系統 LLM | `.env` `LLM_*` 指地端（Kimi 會審查台灣/政府公文）；或 `/admin/models` 選 default.llm |
| GraphRAG | 各 KB 用 `POST /graph/rebuild` 啟用 + 建圖（地端 LLM 抽取）；查詢自動加法融入 |
| 多跳 | 預設關。語料大 + 跨文件關係多時設 `GRAPH_QUERY_HOPS=1`，用 `tools/eval/graphrag_ab.py` 驗有增益再開 |
| 換 embedding | 走 `/admin/models` 套用按鈕（會全庫重嵌，期間搜尋退化）；勿手改 `EMBEDDING_MODEL` env 後不重嵌 |
| nightly 重測 | 有 self-hosted runner 才設 repo variable `ENABLE_NIGHTLY=true` + `NIGHTLY_RUNNER`（GH-hosted 會 OOM）|
