# v4.6 Roadmap — Multi-tenant Account System

Theme **F**：補完整 self-service 帳號流程（接續 v4.1 trial signup、v2.4 OIDC SSO）。

## 範圍

| 項目 | 說明 |
|---|---|
| Email verification | 新註冊 trial / 自助 user 收 verify link，24h 內按掉 → `email_verified_at` |
| Forgot / reset password | `/forgot-password` 寄 token（1h）→ `/reset-password` bcrypt 換 hash |
| OAuth (Google / GitHub) | self-service consumer OAuth，與 v2.4 企業 OIDC SSO 並行不衝突；auto-create user |
| trial.py 補 send verify email | v4.1 留的 TODO 補成 working flow |
| Frontend | VerifyEmail / ForgotPassword / ResetPassword 三 view + LoginView OAuth buttons |

## 不在範圍（v4.x backlog）

- magic link login
- 2FA / TOTP
- 改 chat / agent service

## DB schema

`0016_account_tokens.py`（合進 agent alembic chain，共用同一 PG）：
- `users.email_verified_at TIMESTAMPTZ`
- `users.verify_token VARCHAR(128)` + `verify_token_exp`
- `users.reset_token VARCHAR(128)` + `reset_token_exp`
- partial index on token columns
