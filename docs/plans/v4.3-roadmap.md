# v4.3 Roadmap — Theme C：Plugin SDK + Marketplace

> Status: P1–P4 合併進同一個 PR 落地（第三方寫 custom node / provider / hook）。

## 動機

到 v4.2 為止，workflow executor 內建 ~30 種 node type（v3.5 補完 sub_workflow、kb_writer、human_approval…），provider 系統有 `services/agent/app/core/providers/base.py BaseProvider`。但**所有擴充都要改 monorepo 才能上線**，第三方/客戶完全沒有 plug-in 點。

v4.3 Theme C 目標：開放一條第三方寫 plugin → 包成 Python package → 放 marketplace → admin 一鍵 install 的路徑。

## 範圍（這個 PR）

| 子項 | 範圍 |
|---|---|
| **P1: Plugin SDK** | `packages/python/staffkm-plugin-sdk/` — `BaseNode / BaseProvider / BaseHook` 介面 + `PluginManifest` schema + `runtime.load_plugins_from_dir()` loader |
| **P2: Plugin install API** | `services/agent/app/api/plugins.py` — admin 列出已 load plugin、reload 全部；gateway proxy `/api/v1/admin/plugins` |
| **P3: Marketplace metadata** | `tools/plugin-marketplace/` — README + `index.json` 已知 plugin 清單 |
| **P4: 範例 plugin** | `packages/python/staffkm-plugin-sdk/examples/weather_node/` — `weather_lookup` node 查 wttr.in |

## 不在範圍（v4.4+ backlog）

- **Plugin sandboxing / security**（目前 plugin 跑在 agent 同 process，trust model = workspace admin trusted）
- **Executor 真正 invoke plugin node**：本 PR 只 load + expose registry，`executor.py` 暫不接（避免 break workflow runtime）。整合點在 `app/core/plugin_loader.py` 內標 TODO，留 v4.4。
- **完整 marketplace UI**：admin API 已可列/reload，UI 留 v4.x。
- **Plugin auto-install from git**：本 PR 走 docker volume mount (`./plugins:/app/plugins:ro`)，使用者 `git clone` 後 reload。auto-install 留 v4.4。

## 介面契約

### `BaseNode`
```python
class BaseNode(ABC):
    node_type: str   # 唯一識別
    async def execute(self, config: dict, ctx: PluginContext, inputs: dict) -> dict | AsyncIterator[dict]: ...
```

### `BaseProvider`
```python
class BaseProvider(ABC):
    provider_type: str
    async def chat(self, model: str, messages: list[dict], **kw) -> dict: ...
    async def chat_stream(self, ...) -> AsyncIterator[str]: ...
```

### `BaseHook`
```python
class BaseHook(ABC):
    hook_type: str   # before_run / after_run / before_node / after_node
    priority: int = 100
    async def handle(self, event: dict, ctx: PluginContext) -> None: ...
```

### `PluginManifest` (manifest.yaml)
```yaml
name: my-plugin
version: 0.1.0
nodes: ["plugin:MyNode"]
providers: ["plugin:MyProvider"]
hooks: ["plugin:MyHook"]
requirements: ["httpx>=0.27"]
min_platform_version: "4.3.0"
```
