# v4.1 Roadmap — 客戶 onboarding 自動化

> Theme A：sales demo / trial signup / starter pack
> 估時：1.5 週、4 PRs

## 4 PR 拆解

### PR 1 — Trial signup flow（self-service）
- 新 endpoint `POST /api/v1/public/trial-signup`：email + password → 建 trial workspace + admin user + 14 天 trial 標記
- alembic 0015 `workspaces.trial_expires_at` / `trial_plan`
- 新前端頁 `/signup` (公開、不需登入)
- email verify pending（用 v3.4 SMTP）
- trial expired 自動凍結（trigger 中間 nightly check）

### PR 2 — Starter pack（5 個預建 application template）
- `tools/starter-pack/`：5 個 JSON template
  - HR 助理（請假規則 / 報帳 / onboarding QA）
  - IT helpdesk（密碼重設 / VPN 設定 / 設備借用）
  - 知識庫客服（公司政策 QA）
  - 會議筆記助理
  - 銷售知識庫
- API `POST /api/v1/admin/starter-pack/install`：一鍵安裝到當前 workspace
- 前端 onboarding wizard（v2.3 已有）加「starter pack」 step

### PR 3 — Sales demo data + script
- `tools/sales-demo/`：
  - `seed_demo.py`：建 demo workspace + 預設 user + 上傳 sample docs + 跑 1 個 workflow 留 history
  - `demo-script.md`：銷售展示腳本（10 分鐘 demo flow）
- compose 加 `--profile demo` 自動跑 seed
